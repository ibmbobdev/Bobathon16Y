# 🏛️ AI Government Expense Tracker — Bobathon Lab


You're managing government department expenses. You have a pile of receipts —
office supply invoices, equipment purchases, service contracts — all in different
formats, from different vendors.

Now you have to open each one, figure out what every charge is, type it
into a spreadsheet, and hope you didn't miss anything.

**For a typical monthly review, this takes 30–60 minutes. Every single time.**

---

## What If AI Could Do This In 20 Seconds?

That's exactly what you're going to build in this lab.

Using **IBM Bob** as your AI coding assistant and **IBM watsonx.ai Granite 3**
as the brain, you'll build a web app that:

- 📄 Accepts PDF receipts — office supplies, equipment, services, utilities
- 🤖 Automatically extracts and categorizes every line item
- 📊 Generates interactive charts so you can visualize your spending
- 📝 Writes a plain-English expense summary for you
- 💾 Exports everything to CSV, ready to submit

**You won't write a single line of code by hand.** You'll describe what you
want in plain English, and Bob will build it for you. This is what
AI-assisted development looks like in practice.

---

## The Best Part

By the end of this lab, you'll have a working app you built yourself
from scratch, in under 2 hours, using AI tools that are available to you
right now inside IBM.

## Build a Budget Tracker

Now that your expense tracker is running, take it further with **Lab 2**.

In the bonus lab, you'll use Bob to add a full budget tracking system to
your existing app, no new files needed, just three more prompts.

You'll add:

- 💰 A **total department budget** you can set for the period
- 📂 **Per-category limits** for Office Supplies, Equipment, Services, and Utilities
- 🚨 **Overspend alerts** that flag in red when you've gone over budget
- 📊 A **Budget vs. Actual chart** showing exactly where you overspent

## Want a Better Looking App?

The app works out of the box after Lab 1 — but if you want to take the UI
further, Bob can do that too.

With one additional prompt, you can transform your app from functional to
polished:

- ✨ Animated hero banner with gradient and fade-in effects
- 🎴 Metric cards with hover animations and colored accents
- 🎨 Smooth transitions and hover effects on all buttons
- 📊 Better data table with scrolling and styled headers
- 🤖 AI summary displayed in a gradient highlight box
- 🏷️ Professional footer and cleaner overall layout

No design experience needed. Just paste the prompt into Bob and let it
handle the rest.

👉 **[UI Enhancement Prompt LAB3 →](LAB3.md)**

**How to use Plan Mode:**
Open Bob → switch to **Plan** → describe what you want to add →
Bob will map out the approach before writing any code.
It's great for features you're not sure how to structure yet.
